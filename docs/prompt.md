## 実装の手順

Figma Dev Mode MCP Serverを活用してください。

- 実装対象のスクリーンショットを取得する
- コードを取得する
- Code Connectから既存のコードベースを確認して、既存のコンポーネントと追加実装が必要なコンポーネントを理解する
- デザインのアノテーションから実装対象の動作仕様を理解し、既存のロジック (/src/data/) と追加実装が必要なロジックを理解する
- 実装する
  - UIパーツはStorybookも実装する
